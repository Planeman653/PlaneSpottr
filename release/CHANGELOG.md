# Changelog

All notable changes to PlaneSpottr.

## [0.0.0] - 2026-04-07

### Added

- Initial release of PlaneSpottr
- Real-time flight tracking from FlightAware API
- Dark theme UI
- Configurable search radius and refresh interval
- Standalone .exe installer support
- PyInstaller build configuration
- Rate limiting for API requests

### Features

- Displays nearby flights within configurable radius
- Live flight status updates (en route, delayed, landed, etc.)
- Live flight position, altitude, and ground speed
- Keyboard shortcuts (Ctrl+R, Ctrl+,)
- Settings dialog for preferences

### Built with

- PyQt6
- FlightAware API
- PyInstaller (for .exe builds)
